## Cross-Cutting Changes (done)

These changes span multiple components and are fully implemented:

- **Project Felt theme support** — Added `Theme` enum, `ThemeManager<Theme>`, `pf-v6-theme-felt`. ThemeSelector includes Theme toggle group.
- **Glass mode** — Added `Contrast.GLASS` → `pf-v6-theme-glass`, `Glass`/`NoPlainOnGlass` modifier interfaces, `glass`/`noPlainOnGlass` CSS constants.
- **High contrast mode** — Graduated from beta. `Contrast.HIGH_CONTRAST` → `pf-v6-theme-high-contrast`.
- **Icon system migration** — Replaced FontAwesome with Red Hat brand icons (rhUi, rhMicrons, rhStandard) across all components. Updated `Severity` and `ValidationStatus` enums. Added `rhIcons.mjs` adapter with multi-path SVG flattening.
- **CSS rename** — `no-plain` → `no-plain-on-glass` already using correct naming.
- **Link/nav/control styling** — CSS-only changes, no Java impact.
